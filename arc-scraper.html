ARC Lesson-Plan Scraper

The ARC lesson plans sit behind a Department login, so this has to run inside your own logged-in browser tab. It reads every English literacy skills lesson-plan link on whatever ARC page you are viewing, and gives you a tidy title / id / url / section table you can paste straight into the resources sheet.

How to use it

  1. Log in to ARC and open a lesson-plans listing page (for example the English literacy skills lesson plans index, or any sub-section that lists lessons).
  2. Run the scraper one of two ways:
    • Bookmarklet — drag the red button below onto your bookmarks bar once. Then, on the ARC page, click it.
    • Console — press F12, open the Console tab, paste the snippet below, press Enter.
  3. A panel pops up with the results already selected. Click Copy (or press Ctrl/Cmd+C), then paste into a spreadsheet or send it back here.
  4. Scroll/expand any collapsed sections first if the page hides lessons until opened — the scraper only sees what is currently in the page.
